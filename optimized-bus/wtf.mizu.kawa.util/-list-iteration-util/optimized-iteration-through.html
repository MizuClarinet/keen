open fun <T> optimizedIterationThrough(list: List<T>, consumer: Consumer<T>)

Some List implementations (like java.util.ArrayList) also implement RandomAccess, which benchmarks proved to be faster iterating over using a for-i loop and get. However, non-random-access lists have a better operation/second score using iterator while iterating.

Parameters

<T>

the list item type.

list

the list to be iterated over.

consumer

the consumer to apply on each iteration.
